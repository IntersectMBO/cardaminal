# Update Wallet's Data

After attaching a chain to the wallet, it will be necessary to update the wallet state. This command will get data from chain db only linked with this wallet.

<Callout type="info"> this process can take several minutes </Callout>

## Parameters

Parameters supported by command

- `name`: name to identify the wallet.

## Command

Command description

```sh
cardaminal wallet update [name]
```
