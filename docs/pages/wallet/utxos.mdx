# List Utxos from a wallet

In this section you can list utxos paginated from a wallet.

## Parameters

Below are all parameters supported to list utxos

- `wallet`: wallet name existing
- `page`: number of page to find
- `size`: number of utxo per page

## Command

Command description

```sh
cardaminal wallet utxos [wallet]
```

Below is an example of executing the command

```sh
cardaminal wallet utxos my-wallet
```
