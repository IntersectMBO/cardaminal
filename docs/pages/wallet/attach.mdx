# Attach Chain

To get data from the wallet, a chain will be necessary to attach a chain into wallet.

<Callout type="info"> A chain needs to be created </Callout>

## Parameters

Below are all parameters supported to attach a chain to a wallet

- `wallet`: wallet name existing
- `chain`: chain name existing

## Command

Command description

```sh
cardaminal wallet attach [wallet] [chain]
```
