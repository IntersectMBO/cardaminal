# Attach a chain to a wallet

In this section you can attach an existing chain to a wallet. The wallet and chain must already be configured

## Parameters

Below are all parameters supported to attach a chain to a wallet

- `wallet`: wallet name existing
- `chain`: chain name existing

## Command

Command description

```sh
cardaminal wallet attach [wallet] [chain]
```

Below is an example of executing the command

```sh
cardaminal wallet attach my-wallet my-chain
```
