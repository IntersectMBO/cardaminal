# Dump chain blocks

In this section you can get a group of blocks on a chain.

## Parameters

Below are all parameters supported to get blocks.

- `name`: The chain name. It need will be slug, for example `my-chain-name`
- `skip`: (optional) Number of blocks to skip
- `take`: (optional) Number of blocks to output

## Command

Command description

```sh
cardaminal chain dump [name] --skip [number] --take [number]
```

Below is an example of executing the command

```sh
cardaminal chain dump my-chain-name --skip 10 --take 10
```
