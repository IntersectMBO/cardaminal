# Delete chain

In this section, you can delete a chain. To delete a chain it's necessary to define the chain name parameter. 
> **Warning** All of resource of the chain well be deleted.

## Parameters

- `name`: The chain name. It need will be slug, for example `my-chain-name`

## Command

Command description

`cardaminal chain delete [name]`

Below is an example of executing the command

`cardaminal chain delete my-chain-name`
